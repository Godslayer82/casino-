Welcome to the ultimate lightweight, high-performance virtual arcade. Built entirely inside a **single portable HTML file**, this repository packs 23 fully responsive, interactive casino games featuring math-accurate simulation logic, a neon-drenched grid aesthetic, and persistent balance tracking.

---

## ⚡ Key Architecture Highlights
* **Zero Dependencies:** Pure HTML5, modular Vanilla JavaScript, and advanced CSS Grid layouts. No frameworks, no node modules.
* **Persistent Chips Variable:** Built-in hooks with browser `localStorage` to keep your bankroll saved automatically across page refreshes.
* **Emergency Pity ATM:** Real-time systemic detection checks if your balance hits zero and offers immediate financial injection triggers.
